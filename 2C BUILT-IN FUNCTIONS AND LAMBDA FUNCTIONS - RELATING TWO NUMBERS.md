## BUILT-IN FUNCTIONS AND LAMBDA FUNCTIONS - CALCULATING TWO NUMBERS


### AIM  
To write a Python program to calculate the value of the following expression by using lambda function


### ALGORITHM

1.Start

2.Input values for x, y, and z

3.Define a lambda function:

4.calculate = lambda x, y, z: (x / 10) * (y / 2) * z

5.Call the lambda function with the input values

6.Display the result

7.End



### PROGRAM

exp = lambda x,y,z:(x/10)*(y/2)*z       <br />
x = int(input())                       <br />
y = int(input())                       <br />
z = int(input())             <br />
result = exp(x,y,z)            <br />
print(result)
